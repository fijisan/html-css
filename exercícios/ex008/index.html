Principais formatações:

Negrito / Destaque:

Nessa frase o termo "lua" estará em negrito: Hoje a lua está tão linda!

Isso foi feito usando a tag <b>. (não semântica)

Nessa frase, a palavra "carrapato" estará em destaque: Estou coçando, parece até carrapato, eca!

Isso foi feito utilizando a tag <strong>. (semântica)

Itálico / Ênfase:

Nessa frase, o termo "chocolate" estará em itálico: Estou com vontade de comer chocolate.

Isso foi feito com a tag <i>.

Nessa frase, a palavra "maizena" estará em italico: Minha nossa, como eu amo biscoito maizena!

Isso foi feito usando a tag <EM>.

Texto marcado:

Podemos criar um texto marcado utilizando a tag <MARK>.
Por exemplo: Lorem dolor sit amet consectetur adipisicing elit. Iure, asperiores.

O texto foi marcado com sucesso. É criada uma espécie de marcador de textos.

Texto grande / Texto pequeno:

O termo "calça erguida" estará grande: Corram, lá vem o homem das calça erguida!

Isso foi feito usando a tag <big>.

O termo "Ratinho é foda!" estará pequeno: Devo admitir, o Ratinho é foda!

Isso foi feito utilizando a tag <small>.

O termo "cebola roxa" será indicado como texto excluído: Eu tenho vários amigos que não gostam de cebola roxa repolho.

O texto excluído foi feito com a tag <del>.

Texto inserido:

O termo "calça jeans" será indicado como texto inserido: A nova moda é usar calça jeans chapéu fedora.

O texto inserido foi feito com a tag <ins>.
